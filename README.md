# Legal Automation

Sistema de gestión y automatización de expedientes judiciales.

## Funcionalidades

- Gestión de expedientes, partes y actuaciones
- Generación automática de escritos (Jinja2)
- Persistencia con SQLAlchemy
- Arquitectura modular (models, services, templates)
